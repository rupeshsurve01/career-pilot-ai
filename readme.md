# 📌 Overview

CareerPilot-AI is an intelligent career assistant designed to help job seekers prepare smarter and faster.

Instead of manually analyzing resumes, job descriptions, and interview preparation strategies, CareerPilot-AI uses AI to automate the entire workflow and provide actionable insights.

# ✨ Features

| Feature                   | Description                                |
| ------------------------- | ------------------------------------------ |
| 📝 Resume Optimization    | Improve resume quality & ATS score         |
| 🎯 Interview Intelligence | Generate role-specific interview questions |
| 📊 Skill Gap Analysis     | Detect missing skills                      |
| 📅 7-Day Flight Plan      | Personalized roadmap                       |

# ⚙️ Setup

## Clone Repository

```bash id="jy09eh"
git clone https://github.com/rupeshsurve01/CareerPilot-AI.git
cd CareerPilot-AI
```

---

## Backend Setup

```bash id="gw97de"
cd Backend
npm install
```

Create `.env`

```id="tw2g4g"
PORT=3000
MONGODB_URI=your_mongodb_connection_string
GOOGLE_GENAI_API_KEY=your_gemini_api_key
JWT_SECRET=your_secret_key
```

Run:

```bash id="p0s6uy"
npm run dev
```

---

## Frontend Setup

```bash id="t7y5m0"
cd Frontend
npm install
npm run dev
```
